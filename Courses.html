    Web Development

    Create modern full stack web applications

    Front-End Developer (3 Months)

    In this course you will learn the fundamentals of Front-End Web Development and all the necessary tools to create your first website and Progressive Web Applications (PWA). You will use HTML and CSS to create and style the pages following your design mockups and at the end you will deploy to the world wide web. Afterward you will learn how to create interactive websites using JavaScript and responsive web applications with more advanced frameworks such as React or Vue.

    Full-Stack Developer (4 Months)

    In this course you will learn the same things as the previous course with the following addition. You will be in charge of creating a Back-End using Node.js, either for the application developed during the previous module or for a brand new full stack application. The stack is called M.E.R.N and it consists of the following technologies: M: MongoDB is a non-relational database used to store and access data E: Express is a middleware library used to connect our Back-End with the Front-End R: React is a Front-End JavaScript framework N: Node.js as Back-End JavaScript framework

    Specialization Module (5 Months)

    In this course you will learn the same things as the previous courses with the following addition. You will take part in a collaborative project and you will choose your path depending on the technologies you want to focus on: Gatsby: is a Front-End framework based on React that helps creating static pages pulling data from anywhere (CMS, Markdown, Databases). It has been gaining popularity recently thanks to its great SEO (search engine optimization) capabilities. Typescript: is a JavaScript typed language which you can use in both Front-End (JavaScript, React, Vue) and Back-End. It has always been widely used in companies because it saves you time catching errors and providing fixes before you run your code. In both paths you will also learn how to use GraphQL, the newest query language used in modern start-ups. At the end of the project you will have interview preparation with your mentors and code challenges that will help you in your future job search.     Data Analytics

    Analyse and illustrate data for businesses

    Data Analyst (3 Months)

    This course aims at preparing students to have solid fundamentals of Python and Statistics with a special emphasis on Story telling. Once you grasp the concepts of Pandas and Numpy, you will start mastering Data Visualization Techniques with Matplotlib and Seaborn. You will learn how to deal with Time Series data and implement a Forecast model with the ARIMA, SARIMA along with the latest technologies like Facebook Prophet. Analyzing e-commerce business & marketing insights will help you understand the key concepts of marketing KPIs and its real life use cases. You will learn the fundamental concepts of Machine Learning and apply directly in your projects.

    BI Analyst (4 Months)

    Once you have acquired all the necessary skills of a Data Analyst, we prepare you with business skills and database design. You will learn Tableau/Google Studio, SQL and Machine Learning techniques used in the marketing sector. Tableau: it is the most popular tool for data visualization and creating dashboards. Most of the Fortune Global 500 companies are using this tool in the Business Intelligence department. SQL: is the top language for databases used in the tech industry, especially among data scientists. Is a fundamental knowledge for anyone who wants to enter the world of business intelligence.

    Specialization module (5 Months)

    After harnessing all the skills of a Data/Business Analyst, you will have all the tools to get introduced to either the Data Science world or the Data Engineer world. Based on what you'll pick you will get an overview of the main concepts of Deep Learning and Natural Language Processing or on the main concept of Web Development and Data Pipelines. This last module will then focus on working on your final project, which should reflect the direction you'll want to take in your future career.
